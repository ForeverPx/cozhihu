Getting started on Cozhihu

Question(Q): How to switch zhihu color?

Answer(A): Install chrome extension cozhihu.


Q: what is cozhihu?

A: Cozhihu is a chrome extension for zhihu to switch zhihu page color.


Q: Is cozhihu free?

A: Yes, cozhihu is free for use.


Q: How to faster open zhihu page?

A: You just click the cozhihu icon in any page. If you current page is zhihu page, cozhihu icon is colorful icon which you can swtich the page color. If you click the cozhihu icon and current page is not the zhihu page, that it will navigate you to zhihu home page.
